AdvanceMais API

Plataforma de serviços integrados para advogados e escritórios jurídicos

Status da API: Operacional
Versão: 1.0.0

Sobre a API

A API do AdvanceMais fornece acesso programático aos recursos da plataforma, permitindo a integração com sistemas jurídicos, gerenciamento de usuários, empresas, pagamentos e conteúdo do site.

Principais recursos:

  • Autenticação segura via JWT
  • Gerenciamento de usuários e empresas
  • Processamento de pagamentos
  • Sistema de CMS para o site
  • Notificações por e-mail e SMS
